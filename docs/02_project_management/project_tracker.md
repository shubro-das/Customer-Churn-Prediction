# Project Tracker

## Purpose

The Project Tracker is the central project management document for the Bank Customer Churn Prediction Platform.

Its purpose is to monitor project progress, track responsibilities, manage milestones, and ensure that every planned deliverable is completed. Unlike the project roadmap, which defines the overall direction of the project, this tracker is updated continuously throughout development.
